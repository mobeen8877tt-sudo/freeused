# Build a Modern AI-Style QR Code Generator Website

Create a complete, production-ready, SEO-optimized website for a **Free QR Code Generator**. The website should have a premium AI-inspired futuristic design with smooth animations, modern UI, and excellent user experience.

## Main Goal

Users should be able to generate QR codes for free without creating an account.

## Core Features

* Generate QR codes instantly.
* No login or registration required.
* Unlimited free QR code generation.
* Fast QR code generation.
* High-resolution output.
* Download QR codes in:

  * PNG
  * JPG
* Responsive on desktop, tablet, and mobile.
* Dark mode and light mode.
* Copy generated QR code link.
* One-click download.
